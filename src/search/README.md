# LegalSemanticSearchEngine

Движок для семантического поиска и тегирования юридических текстов (статей Конституции).

## Описание
Класс `LegalSemanticSearchEngine` предназначен для автоматического присвоения научно обоснованных тегов статьям и поиска релевантных статей по запросу пользователя на основе семантической близости.

## Математическая модель
Движок использует векторные представления (embeddings) для оценки семантической близости:

1.  **Эмбеддинги**: Тексты (статьи и теги) преобразуются в векторы $v \in \mathbb{R}^d$ с помощью модели `SentenceTransformer`. Векторы нормализуются: $\|v\|_2 = 1$.
2.  **Косинусное сходство**: Сходство между двумя векторами $a$ и $b$ вычисляется как скалярное произведение:
    $$\text{similarity}(a, b) = \frac{a \cdot b}{\|a\| \|b\|} = a \cdot b \quad (\text{так как векторы нормализованы})$$
3.  **Ранжирование статей**: Для запроса $q$ и статьи $s$ с набором тегов $T_s$ и весами $W_s$, итоговая оценка $S$ рассчитывается с учетом общих тегов $T_{common} = T_q \cap T_s$:
    $$S = \text{cosine\_similarity}(v_q, v_s) + \text{bonus} - \text{penalty}$$
    где:
    - $\text{bonus} = 0.2 \cdot \frac{|T_{common}|}{N}$
    - $\text{penalty} = 0.2 \cdot \frac{|T_q \setminus T_s|}{N}$
    - $N$ — максимальное количество тегов на статью.
    Результат ограничивается диапазоном $[0, 1]$.

## Основные возможности
- **Автоматическое тегирование**: Присвоение тегов статьям на основе семантического сходства.
- **Семантический поиск**: Поиск статей по текстовому запросу пользователя.
- **Кэширование**: Поддержка кэширования моделей и вычисленных эмбеддингов.

## Основные методы

### `assign_tags(articles_list, tags_per_article)`
Присваивает теги списку статей, используя косинусное сходство между эмбеддингами статей и тегов.

### `get_tag_recommendations(text)`
Возвращает словарь `{тег: оценка_релевантности}` для заданного текста.

### `find_articles_by_new_sentence(new_sentence, k=5)`
Находит топ-k наиболее релевантных статей, используя описанную выше математическую модель ранжирования.

## Использование
```python
from src.search.engine import LegalSemanticSearchEngine

# Инициализация
engine = LegalSemanticSearchEngine(tags_filepath="data/raw/base.json")

# Поиск статей
results = engine.find_articles_by_new_sentence("Ваш запрос", k=5)
```
